# 🚀 Panaversity Assignment Projects

This repository contains three projects completed as part of a Panaverse assignment. The projects focus on AI prompting, JavaScript game development, and AI-powered slide generation.

## 📂 Projects

### 🧠 AI Prompting Lab
A hands-on practice project covering 13 prompt engineering concepts using different AI models.

**Topics Covered:**
- Context-based prompting
- Reasoning and decision-making
- AI search and research
- Prompt iteration
- Data analysis
- Model comparison

### 🎮 Snake Game
A classic Snake Game built with HTML, CSS, and JavaScript.

**Features:**
- Keyboard controls
- Score tracking
- Collision detection
- Interactive gameplay

### 📊 LLM Slide Generator
An AI-powered tool that generates presentation slides from input content.

**Features:**
- Content summarization
- Slide outline generation
- Automated presentation structure

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- ChatGPT
- Claude
- Google AI Studio

## 🎯 Learning Outcomes

- Understanding prompt engineering techniques
- Building logic through JavaScript projects
- Using LLMs for content generation and automation
- Comparing outputs from different AI models
